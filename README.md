FunWave is Swift3.+ simple fun wave open source component  

Require
==============
* iOS 8.0+ / Mac OS X 10.11+ / tvOS 9.0+
* Xcode 8.0 or later
* Swift 3.0+

Install
==============
* CocoaPods: pod 'FunWave'

Usage
==============
```swift
let wave = FunWaveView()
/// set frame/autolayout .....
wave.start()
/// if stop wave animation call stop() method
```
